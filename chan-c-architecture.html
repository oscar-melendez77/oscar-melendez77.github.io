Vendor Seams — Swap Without Touching Business Logic
TwilioProvider ———▶ MetaProvider
MESSAGING_PROVIDER=meta in .env
Same interface for list/button/template sends.
Migrate Twilio → Meta Cloud API, zero caller changes.
CertificadorClient ———▶ INFILEClient
CERTIFICADOR=infile in .env
FEL issuance behind one client interface.
Swap SAT-authorized certificador without refactors.
Provider-Seam Pattern
# config/providers.py — swappable vendors from day one

MESSAGING_PROVIDER = os.getenv("MESSAGING_PROVIDER", "twilio")  # flip to "meta"
CERTIFICADOR = os.getenv("CERTIFICADOR", "infile")

def get_messenger():
    if MESSAGING_PROVIDER == "twilio": return TwilioProvider()
    if MESSAGING_PROVIDER == "meta":   return MetaProvider()  # business logic never changes
